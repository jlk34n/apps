Discovery Health PICO Search Widget

The EBSCO Health PICO widget is intended to help users formulate searches using the PICO framework. The widget assembles the terms entered into each of the fields to form an EDS search expression. The search expression is transfered to the EDS search dialog, and the Search (submit) button is activated automatically.

There is evidence [1] that using the PICO framework can increase the precision of searches in a clinical setting. A number of medical libraries and database providers have made PICO tools available to their end users, as well as research guides and instructions on the use of the PICO framework. Some useful examples:

Using the EDS PICO widget

The EDS PICO widget is a simple tool that provides users a form for parsing their search terms according to the PICO framework, and mapping those terms to an EDS search expression.

  • Required Fields: Population/Problem, Intervention
  • Optional Fields: Comparison, Outcomes

  • Submit: users must click on the submit button to execute the PICO search in EDS
  • Reset: clears the form
  • Help: displays this page

Recommendations

  1. Start your search using broad or simplified terms to describe a condition or intervention; in many cases, these will be automatically mapped to the more specific descriptors in each database.
  2. PICO searches should be focused on matters of therapy, diagnosis, or prognosis. [2] Use the EDS Basic or Advanced search for basic research questions.
  3. Do not include Boolean operators in your PICO search (and, or).
  4. Once you've submitted your initial search, use the EDS facet and limiter tools to further refine or expand your search results. Remember, PICO is a tool to get you started, but there are many other tools within EDS that help you refine your results.
  • [1] Utilization of the PICO framework to improve searching PubMed for clinical questions; BMC Medical Informatics and Decision Making. 2007 7:16 | DOI: 10.1186/1472-6947-7-16
  • [2] Richardson, et al: The well-built clinical question: a key to evidence-based decisions. ACP Journal Club, Nov/Dec 1995, A12-13
